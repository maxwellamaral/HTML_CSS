CSS
Cascading Style Sheets (CSS) é um mecanismo para adicionar estilo (cores, fontes, espaçamento etc.) a um documento web
HTML
Abreviação para a expressão inglesa HyperText Markup Language, que significa: "Linguagem de Marcação de Hipertexto". É uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.
